# 📊 Financial KPI Analysis for a Startup

##  Project Overview

This project analyzes the financial performance of a startup using Key Performance Indicators (KPIs). The analysis was completed using Microsoft Excel and Python (Pandas). The project demonstrates how financial data can be used to evaluate business growth, customer acquisition efficiency, and revenue trends.

---

##  Objectives

- Analyze monthly financial performance.
- Calculate important financial KPIs.
- Visualize financial data using charts.
- Understand startup growth through KPI analysis.

---

##  Tools Used

- Microsoft Excel
- Python
- Pandas
- Visual Studio Code

---

##  Financial Dataset

The dataset contains six months of startup financial data with the following fields:

- Month
- Revenue
- Expenses
- Marketing Cost
- New Customers
- Average Revenue per Customer
- Customer Lifetime

---

##  Key Performance Indicators (KPIs)

The following KPIs were calculated:

- Burn Rate
- Customer Acquisition Cost (CAC)
- Lifetime Value (LTV)
- LTV:CAC Ratio
- Run Rate

---

##  Dashboard

An Excel dashboard was created to visualize:

- Monthly Revenue Trend
- Monthly Expenses Trend
- Burn Rate
- Customer Acquisition Cost (CAC)
- Lifetime Value (LTV)
- Run Rate

---

##  Python Implementation

Using the Pandas library, the project:

- Created the financial dataset
- Calculated all KPI values
- Displayed the analyzed dataset

Run the program using:

```bash
python financial_kpi.py
```

##  Results

- Revenue increased steadily from January to June.
- Expenses increased gradually.
- Customer Acquisition Cost (CAC) remained efficient.
- Lifetime Value (LTV) remained constant.
- LTV:CAC Ratio showed healthy customer profitability.
- Run Rate increased with monthly revenue growth.

---

##  Conclusion

This project demonstrates how Microsoft Excel and Python (Pandas) can be used to analyze startup financial data and calculate key business metrics. The results provide valuable insights into business performance and support better financial decision-making.
